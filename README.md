# Ali Aliyev - Personal Portfolio Website

## About This Project

This is a personal portfolio website built as part of the SITE 1101: Principles of Information Systems course. The site showcases my journey as a competitive programmer, my achievements in Informatics Olympiads, and my passion for machine learning and education.

## Features

- **Responsive Design**: Mobile-first approach, fully functional on all device sizes
- **Modern Aesthetic**: Inspired by ciccarello.me design patterns
- **Smooth Scrolling Navigation**: Seamless navigation between sections
- **Scroll Animations**: Elements animate into view as you scroll
- **Professional Layout**: Clean, organized sections for showcasing work and achievements
- **Social Media Integration**: Direct links to all my profiles

## Website Sections

1. **Home (Hero)**: Introduction with profile picture and call-to-action buttons
2. **About**: Biography, education, languages, and contact information
3. **Skills**: Technical skills in programming, competitive programming, and tools
4. **Achievements**: Olympiad medals, awards, and research publications
5. **Projects**: Featured projects including digital logic gates and competitive programming work
6. **Connect**: Social media and contact links
7. **Footer**: Copyright and course attribution

## Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Modern styling with CSS variables, Grid, Flexbox
- **JavaScript**: Interactive features and scroll animations
- **Git**: Version control and repository management
